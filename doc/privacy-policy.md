# Privacy Policy

**Last updated: 2026-04-10**

## Overview
Pokemon Raider is a Discord bot that allows users to read and edit 
Google Sheets through Discord commands.

## Data We Access
- Google account identity (name, email) for authentication purposes
- Google Sheets data that users explicitly authorize

## How We Use Your Data
- Authentication data is used solely to identify the user's Google account
- Sheets data is accessed only when triggered by user commands
- No data is stored on our servers beyond the active session

## Data Sharing
We do not sell, trade, or share your data with third parties.

## Data Retention
No user data is persistently stored by this application.
